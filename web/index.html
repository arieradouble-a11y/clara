Clara

Turn complex text into verified plain language. Numbers, dates, and negations are checked so nothing is silently dropped or flipped.

Runs on the server's configured model. With the default mock provider it echoes the text — set CLARA_PROVIDER for real rewriting.

No model needed — this compares the two texts and works fully offline.
